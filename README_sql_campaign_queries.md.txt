# Marketing SQL Practice

This repository contains beginner-to-intermediate SQL queries for analyzing digital marketing campaign performance. The goal is to practice common metrics used in advertising analytics, including CPM, CTR, and ROAS.

## Metrics Explored

- **CTR** (Click Through Rate)  
- **CPM** (Cost per 1,000 Impressions)  
- **ROAS** (Return on Ad Spend)  
- Channel-level performance comparison

## Use Cases

- Identify high-performing campaigns  
- Compare performance across advertising channels  
- Optimize marketing spend allocation

## Tools

- SQL (for querying campaign datasets)
- Practice queries on hypothetical ad campaign data

This project is part of my efforts to develop practical skills in marketing analytics and data-driven decision making.
